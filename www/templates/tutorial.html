Tutorial

There are three properties


1) Shape (Circle, Triangle, Square)

2) Colour of the shape (Red, Blue, Yellow)

3) Colour of the background (White, Black, Grey)


Click to examine each of the three properties

The magic rule of 3


Pick 3 squares from a 3x3 grid, where each of the 3 properties are all either all different or all the same.


Example of a possible valid combination:
1) "same" shape
2) "same" backgrond colour
3) "different" colour of the shape


Practice #1


Is the following a valid combination ?

Yes No

Ooops, this is actually an invalid combination.

1) The shapes are neither all different nor all the same  

2) The colour of the shapes are all the different

3) However, the colour of the background are same  


Great ! You got it.


Practice #2

1) The shapes are all different
